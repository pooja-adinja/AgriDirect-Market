# AgriDirect - Minimal Prototype
This is a minimal prototype of the AgriDirect project:
- Flask backend (`backend/app.py`)
- SQLite database (`backend/agri.db`)
- Simple frontend (`frontend/index.html`) using fetch to call API

How to run (backend):
1. Create and activate a Python environment.
2. Install requirements:
   `pip install -r backend/requirements.txt`
3. Run backend:
   `python backend/app.py`
4. Open `frontend/index.html` in a browser (or serve it with a static server).
